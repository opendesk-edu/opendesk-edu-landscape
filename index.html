About This Landscape

The openDesk Edu Landscape is an interactive map of the 25 integrated open-source services that make up the openDesk Edu platform. Inspired by the CNCF Landscape, this visualization helps institutions, IT administrators, and developers understand the complete ecosystem.

How to Use

  • Search for specific services, categories, or tags
  • Filter by service tier (Critical, High, Standard, Low)
  • Click on any service to visit its website or repository
  • Browse by category to understand the ecosystem structure

Contribute

This landscape is open-source and community-driven. To add a new service, update information, or suggest improvements, please open a pull request on GitHub. The data is stored in data/services.yaml and can be edited easily.

Tier Classification

  • Critical Tier (8 services): Foundation services with 99.9% availability targets (e.g., Keycloak, Nextcloud, email infrastructure)
  • High Tier (7 services): Important services with 99.5% availability (e.g., LMS, video conferencing, helpdesk)
  • Standard Tier (8 services): Collaboration tools with 99.0% availability (e.g., Kanban, surveys, documentation)
  • Low Tier (2 services): Supporting tools (e.g., stateless diagram editors)
